## How to use this index

- Add each new podcast as a row in the appropriate table (or create a new category section).  
- Keep titles short and descriptive; use the **Tags** column for quick filtering.  
- When a podcast has multiple episodes, either:
  - Link the show home page once and note “multi‑episode” in Notes, or  
  - Add one row per key episode.
