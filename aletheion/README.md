## Vision
Aletheion is a non-fictional, deployable smart-city architecture designed for Phoenix, Arizona. It integrates environmental sensing, energy mesh-grids, waste circular economy, transport mobility, agriculture, health privacy, governance, and security into a unified, offline-capable system. All operations enforce Indigenous sovereignty (Akimel O'odham/Piipaash), neurorights, and ecological stability via computationally hard constraints.

## Core Principles
1. **Sovereign Operability:** Indigenous land rights and data autonomy are enforced via ALN contracts.
2. **Offline-First:** All subsystems function ≥72 hours without network connectivity (Haboob/Grid Failure).
3. **Zero-Knowledge:** Citizen data is encrypted locally; no behavioral prediction or advertising.
4. **Ecological Stability:** Lyapunov residuals ($V_t$) ensure actions never degrade environmental health.
5. **Post-Quantum Security:** No blacklisted algorithms (SHA-256, BLAKE, etc.); lattice-based cryptography only.

## Installation
1. **Clone:** `git clone https://github.com/Doctor0Evil/Aletheion.git`
2. **Verify:** Run `lua deployment/installer/src/city_installer.lua` (Offline-Capable).
3. **Configure:** Edit `aletheion/config/phoenix_profile.json` for local coordinates.
4. **Deploy:** Installer verifies Indigenous territory, creates backups, and deploys subsystems.
