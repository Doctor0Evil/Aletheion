## Overview

This Android client provides augmented citizens with secure access to their Evidence Wallets and Clinical BCI interfaces. It enforces neurorights, consent, and equal protection at the device level.

## Features

- **Evidence Wallet:** View health and eco improvements with ROW references.
- **Clinical BCI Manager:** Securely manage BCI connections with safety kernel enforcement.
- **Neurorights Protection:** Explicit consent flows and prohibited action blocking.
- **Consciousness Preservation:** Status display and request logging (requires Clinical Safety Board approval).
